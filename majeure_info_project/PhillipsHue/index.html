PhillipsHue controller



This is the website create to manage several PhillipsHue.
This is one of the front-ends of the project. The project consist in manage one PhillipsHue from a web front-end (this one), from a android front-end or from connected objects (buttons or other sensors).
The states of the lamp is stored in a database managed by a java springboot back-end deployed on Heroku
The lamp, the fronts and the database are synchronized
